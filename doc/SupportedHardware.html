ADI BSP Supported Hardware

Almost any ADI hardware that has an IIO supported driver can stream data into MATLAB or Simulink. We have specific implementations of the following devices, boards, or chips that support streaming:
  • ADRV9361-Z7035
  • ADRV9364-Z7020
  • AD-FMCOMMS2-EBZ
  • AD-FMCOMMS3-EBZ
  • AD-FMCOMMS4-EBZ
  • ADRV9009-W/PCBZ
  • ADRV9371-W/PCBZ
  • ADRV9371-N/PCBZ

Supported Targetable Hardware

  • ADRV9361-Z7035
  • ADRV9364-Z7020
  • AD-FMCOMMS2-EBZ with Xilinx Zedboard / ZC702 / ZC706
  • AD-FMCOMMS3-EBZ with Xilinx Zedboard / ZC702 / ZC706
  • AD-FMCOMMS4-EBZ with Xilinx Zedboard / ZC702 / ZC706
  • AD-FMCOMMS5-EBZ with Xilinx ZC706
  • ADRV9371-N/PCBZ and ADRV9371-W/PCBZ with Xilinx ZC706 / ZCU102
  • ADRV9009-W/PCBZ with Xilinx ZCU102

System Requirements

Specific examples will require different toolboxes. The targeting examples will require:
  • HDL Coder
  • Embedded Coder (Only for External Mode and ARM targeting)
  • Simulink Coder
  • HDL Verifier (Only for FPGA Capture Demo)
